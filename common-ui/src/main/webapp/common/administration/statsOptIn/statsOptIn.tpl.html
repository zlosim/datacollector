Improve Data Collector with Usage Statistics

StreamSets would like to collect anonymized usage statistics about features used, system performance, etc. This information helps us understand how users actually use the product in order to improve product performance and guide decisions regarding future development of the product. As with all development efforts, improvements to open source products help the whole community.

StreamSets is committed to protecting user privacy. Collection of these usage statistics will not include the collection of any sensitive user information, such as stage configuration details, pipeline names, credentials, etc. No personally identifiable information will be collected. StreamSets will use these anonymized usage statistics for its own internal development of the product and support of its users. Information about usage statistics may be shared with the community on an anonymized aggregated basis in the form of blog posts and other published information about how the product is commonly used.
