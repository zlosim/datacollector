Working with DPM

Before you can work with DPM, you must have a DPM organization and user account.

To work with DPM, you complete the following tasks within the Data Collector UI:

  • Register Data Collector to work with DPM.
  • Configure pipelines to aggregate statistics.
  • Manage how pipelines work with DPM.
  • Optionally, modify DPM configuration properties.
  • Optionally, unregister Data Collector from working with DPM.
